Introduo

A Doena Renal Crnica ou Insuficincia Renal Crnica uma patologia bastante agressiva que provoca a perda irreversvel das funes renais. Esta patologia tem uma elevada taxa de mortalidade. [1]
Como tal, torna-se importante perceber quais os fatores de risco que esto associados a tal doena, nomeadamente, a hemoglobina, a doena arterial coronria, anemia, entre outros.

Assim, a Chronic_Kidney_Disease DataSet rene vrias informaes referentes a possveis fatores de risco assim como consequentes problemas, relacionados com o estgio inicial da Doena Renal Crnica. Estes dados foram recolhidos na ndia, em julho de 2015, com o apoio da Universidade de Alagappa.

Na anlise realizada apenas foram considerados 12 atributos, que iro ser de seguida descriminados, com o propsito de realizar uma dissecao mais restrita.

Alteraes Efetuadas ao DataSet

Como podemos constactar, o DataSet escolhido possui muitos valores em falta (NA). Contudo, para que a anlise pretendida fosse clara e objetiva, tornou-se necessrio remover estes valores.

#Ocultar as ocorrncias com valor 'NA'

#N de Ocorrncias depois de retirados os 'NA'

Dado que, depois de eliminar os valores em falta, o novo DataSet ficou com 158 ocorrncias, no foi necessrio fazer uma seleo randomizada dos dados.

#Apresenta o nmero total de Outliers dos Atributos Contnuos, assim como a sua Mdia
conta.med <- function(df)
{
  for(i in 1:ncol(df))
  {
    #Verificao se o atributo  numrico
    if(is.numeric(df[,i]))
    {
      #Obteno dos outliers 
      out <- boxplot.stats(df[,i])$out
      #Clculo da Mdia 
      med <- mean(df[,i])
      print(paste(names(df)[i], " : outliers -",length(out)," | mdia - ", round(med,2) ))
    }
  }
}
